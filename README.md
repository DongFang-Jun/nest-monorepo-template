## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
$ pnpm start.ts:all
```

## 新增微服务

```bash
# 第一步
$ nest g app base

# 第二步
$ package.json增加启动命令

# 第三步
$ config 新增base相关配置文件

# 第四步
$ 网关服务app.module.ts引入新增微服务名BASE
```

## 新增lib

```bash
$ nest generate library my-lib
```

## 微服务下新增模块

```bash
# 第一步
$ cd apps/base/src

# 第二步
$ nest g module payment

# 新增service
$ nest g service payment

# 新增controller
$ nest g controller payment
```

## 常见问题

#### 1.新增微服务后，请求接口网关服务通畅，微服务中未接收到请求？

```bash
# 第一步
$ cd apps/base/src

# 第二步
$ nest g module payment

# 新增service
$ nest g service payment

# 新增controller
$ nest g controller payment
```
